# Coffeedrop frontend

[View Live Here]()

Built with:

- Vue JS 2.6.11
- Bootstrap-vue 2.17.3
- Sass

## How to use

### Clone the repo

```bash
  git clone
```

### change directory

```bash
  cd
```

### Install dependencies

```bash
  npm install
```

### Create `.env` file from `.env.example`

```bash
  cp .env.example .env
```

- Add your api key for use of HERE maps
- Add url for locations api

### Run the dev server

```bash
  npm run serve
```

- Project will be served to `localhost:8080`
